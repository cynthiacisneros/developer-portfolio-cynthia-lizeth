# Interation-Plan

## P01 Evaluation

Overall, my Project 01 has a solid foundation. The structure of the site is clear, and the main pages function as intended. I also think the overall design direction works well—the soft, minimal aesthetic feels cohesive and appropriate for the audience I had in mind. Some sections, especially those meant to guide or inform the user, feel intentional and well thought out.

However, there are areas that need improvement. The About Me section feels a bit weak in terms of writing and structure. The paragraphing could be stronger, and the content does not communicate as clearly or confidently as I would like. It doesn’t fully reflect who I am or what I want the user to take away from it. I also think the photos section may not be necessary. Right now, it doesn’t add much value to the experience and feels more like extra content rather than something purposeful.

In addition, there are some layout issues throughout the site. Certain sections feel too boxed in, and spacing is not always consistent, which affects the overall flow. There are also small inconsistencies in headings and formatting across pages.

Compared to my original proposal, I did follow through on the main idea and structure of the site. However, some parts were simplified, and others were not as refined as I initially planned. My focus for P01 was getting a complete and functional version, and now my goal is to strengthen the content and overall presentation so it feels more finished and intentional.

##  Changes for P02

## Fixes
I will fix spacing and alignment issues across pages to create a more consistent and polished layout. This will improve the overall visual flow of the site.
I will correct inconsistencies in headings and formatting so that all pages feel unified and intentional.
Improvements
I will revise the About Me section to strengthen the writing, improve paragraph structure, and make the content more clear and purposeful. This is important because it represents who I am and sets the tone for the site.
I will improve the overall layout by reducing the “boxed-in” feeling and allowing sections to have better spacing and balance.
I will continue refining responsiveness using DevTools to ensure the site works well across different screen sizes.


## Additions
I will enhance the projects section by making it more informative and structured. This includes clearer descriptions, tech stack labels, and direct links to both the live site and GitHub code.
I may add subtle hover effects to improve interactivity, while keeping the design clean and minimal.


## Cuts
I am considering removing or redesigning the photos section, as it currently does not add meaningful value to the user experience. If it remains, it will need to be more intentional.
I will avoid adding unnecessary features that do not support the main purpose of the site, so I can focus on strengthening what is already there.

##  Updated Tools and Approach

For Project 02, I will continue using HTML, CSS, and JavaScript, since they provided a strong foundation for building my site. At this stage, my focus is on refining and improving what I have already built rather than introducing new technologies.

Visual Studio Code was an effective tool during P01, especially with extensions and its built-in Git features. Using GitHub also helped me manage changes and track progress. I will continue using this workflow because it supports how I like to work and keeps my process organized.

If I were starting over, I would spend more time planning my content and layout before building. This would have helped me avoid revisiting and revising those areas later.
