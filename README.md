This project is a personal portfolio website built to showcase my work, skills, and growth as I transition into a website developer. It highlights projects I’ve completed, what I’ve learned from them, and the tools I’m actively developing skills in.

This portfolio is part of my coursework and will continue to evolve as I build more projects and gain experience.

## Site structure

- **Live site:** open **`index.html`** — single scrolling page (hero, about, skills, projects, contact) with in-page navigation.
- **Shared assets:** `style.css`, `script.js`, and `images/`.
- **Archived multi-page HTML:** previous standalone pages are kept under **`archive/legacy-html/`** with paths adjusted so they still load root CSS/JS/images when served from the repo root (see that folder’s `README.md`).
